# 🏯 Gateway of Regrets: Soul Defense (해원문)

> **解冤門** — 원혼의 한을 풀어주는 문

---

### 🎮 Overview

Gateway of Regrets는 한국 전통 설화와 민속에서 영감을 받은 타워 디펜스 RPG입니다. 플레이어는 해원문(解冤門)을 지키며, 원혼들의 한을 풀어주는 수호자가 됩니다.

> 한국 설화 기반 타워 디펜스 RPG

**"원한이 깊으면 귀신이 되고, 한을 풀어주면 꽃이 핀다."**

---

## 🎮 게임 소개

해원의 문은 한국 전통 설화와 민속에서 영감을 받은 타워 디펜스 RPG입니다. 플레이어는 해원문(解冤門)을 지키며, 원혼들의 한을 풀어주는 수호자가 됩니다.

### 핵심 시스템

| 시스템               | 설명                                                              |
| -------------------- | ----------------------------------------------------------------- |
| ⚔️ **타워 디펜스**   | 궁수, 병영, 마법 타워 3종 (각 2갈래 분기 업그레이드)              |
| 🌙 **낮/밤 사이클**  | 밤에는 영혼형 적 회피율↑, 타워 범위↓                              |
| 😢 **한(恨) 게이지** | 적이 오래 생존하면 한 게이지↑ → 타워 공속↓                        |
| 👻 **원혼의 순환**   | 적 처치 시 원혼 스폰 → 수집하여 자원 획득 or 적이 흡수하면 광폭화 |
| 👥 **영웅 시스템**   | 5명의 한국 설화 영웅 (깨비, 미호, 강림, 수아, 바리)               |

### 챕터

1. **굶주린 자들의 장터** — 아귀, 짚신귀신, 처녀귀신
2. **통곡하는 숲** — 호랑이 하인, 불개, 이무기
3. **얼굴 없는 숲** — 새타니, 그림자 아이, 달걀귀신

## 🚀 실행 방법

### 요구사항

- Flutter SDK 3.x 이상
- Dart SDK ^3.6.0

### 실행

```bash
# 의존성 설치
flutter pub get

# Windows 실행
flutter run -d windows

# 또는 배치 파일
.\start_game.bat
```

### 빌드

```bash
flutter build windows --release
```

## 🏗️ 프로젝트 구조

```
lib/
├── common/          # 상수, 열거형
├── data/            # 게임 데이터 (적, 타워, 웨이브)
├── game/
│   ├── components/  # Flame 컴포넌트 (적, 타워, 영웅)
│   ├── systems/     # 게임 시스템 (웨이브, 원한)
│   └── world/       # 맵, 낮밤 사이클
├── state/           # Riverpod 상태관리
├── ui/              # Flutter UI (HUD, 메뉴)
└── main.dart        # 진입점
```
