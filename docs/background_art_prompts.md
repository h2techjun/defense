## 🗺️ II. 에피소드 맵 배경 설계 전략

### 접근 방식: **에피소드당 1기본 + 변형 3종**

> **권장**: 에피소드당 20스테이지를 모두 다른 배경으로 만들면 **80장** (시즌1 기준).  
> 현실적으로 **에피소드당 기본 배경 1장 + 시간/날씨 변형 3종 = 4장** 씩 제작합니다.

| 변형         | 설명                             | 적용 스테이지 |
| ------------ | -------------------------------- | ------------- |
| ☀️ 낮 (기본) | 밝은 기본 버전                   | 1-1 ~ 1-10    |
| 🌅 석양      | 붉은 톤 전환                     | 1-11 ~ 1-15   |
| 🌙 밤        | 어둡고 영혼 불꽃 추가            | 1-16 ~ 1-19   |
| 🔴 보스      | 붉은 안개 + 균열 + 강렬한 분위기 | 1-20 (보스전) |

> **총 필요 이미지**: 에피소드 5 × 변형 4 = **20장** (시즌1 전체)

각 에피소드별:
