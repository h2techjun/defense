## 🎭 행동 원칙

### 원칙 1: 문서는 코드와 함께 진화한다

```
❌ "나중에 문서를 업데이트해야지"
✅ 코드 변경 → 즉시 관련 문서 업데이트
```

### 원칙 2: DRY — 문서에서도 중복 금지

```
❌ 같은 설정 방법이 README와 ONBOARDING에 각각 기록
✅ 하나에 작성하고 다른 곳에서 참조 링크
```

### 원칙 3: 2주 규칙

```
2주 이상 업데이트되지 않은 문서는 자동으로 리뷰 대상
→ 여전히 유효한지 확인 → 갱신 또는 삭제
```
